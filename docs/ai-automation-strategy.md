# AI Automation Strategy — Running The Grand Tour on Autopilot

**Date:** 2026-03-20
**Premise:** Automate everything except final approval decisions. AI agents handle marketing, content, support, operations, and even product iteration. The human (Haris) approves, steers, and collects revenue.

---

## The Vision: A One-Person, AI-Operated Business

Your role reduces to:
1. **Approve** — review AI-generated content/campaigns before they go live (15 min/day)
2. **Steer** — make strategic decisions when the AI flags options (1 hr/week)
3. **Collect** — monitor revenue dashboard, handle payouts

Everything else is automated. Here's how.

---

## 1. Content Marketing Engine (Fully Automated)

### The SEO Blog Machine

**What it does:** Generates 3-5 SEO-optimized travel articles per week, targeting long-tail keywords that drive organic traffic to the app.

**How it works:**

```
[Weekly Cron]
  -> Claude Agent: research trending travel keywords + search volume
  -> Claude Agent: write 1,500-word article with destination tips, local phrases, hidden gems
  -> Claude Agent: generate meta description, title tags, internal links
  -> Push to CMS (Ghost, WordPress, or static site) as draft
  -> Slack/email notification to Haris: "5 articles ready for review"
  -> Haris: skim titles, approve batch
  -> Auto-publish on schedule (Mon/Wed/Fri)
```

**Example articles the AI writes:**
- "The Perfect 7-Day Anniversary Trip to Tuscany (2026 Guide)"
- "12 Italian Phrases Every Couple Should Learn Before Their Trip"
- "Rome to Amalfi: A Romantic Road Trip Itinerary"
- "How to Plan a Surprise Anniversary Trip (Without Your Partner Finding Out)"

**Stack:** n8n workflow (self-hosted, free) + Claude API + Ghost CMS ($9/mo) or markdown to Vercel blog (free)

**Cost:** ~$0.15/article in API tokens. 20 articles/month = $3/month. The ROI on organic traffic is essentially infinite.

### Social Media Autopilot

**What it does:** Posts daily across Instagram, TikTok, Pinterest, X, and Facebook — all generated by AI, all on-brand.

**How it works:**

```
[Daily 6 AM Cron]
  -> Claude Agent: pick today's content theme from rotation
     (destination spotlight / Italian phrase / user testimonial / travel tip / app feature / countdown)
  -> Claude Agent: write platform-specific copy
     - Instagram: visual caption + 30 hashtags
     - TikTok: hook-first script for text overlay video
     - Pinterest: pin title + description + keywords
     - X: punchy one-liner + thread option
     - Facebook: conversational post
  -> Image generation: Gemini or DALL-E generates a destination photo / quote card / app screenshot
  -> n8n pushes to Buffer/Later/native APIs
  -> Posts go live at optimal times per platform (Buffer handles scheduling)
```

**Content rotation (7-day cycle):**
| Day | Theme | Example |
|-----|-------|---------|
| Mon | Destination Spotlight | "Pienza: The Renaissance 'Ideal City' you've never heard of" |
| Tue | Italian Phrase | "Say 'Cin cin!' (chin chin) — the Italian way to toast" |
| Wed | App Feature Demo | 15-sec screen recording of the stamp collection moment |
| Thu | Travel Tip | "ZTL zones in Italy: What every driver must know" |
| Fri | Couple Story | "How Sarah surprised her husband with a customized trip app" |
| Sat | Weekend Wanderlust | Stunning landscape photo + "Where would you go?" engagement post |
| Sun | Behind the Scenes | "I built this app as an anniversary gift. Here's the story." |

**Stack:** n8n + Claude API + Gemini (images) + Buffer ($6/mo for 3 channels) or native APIs

**Cost:** ~$0.30/day in API tokens + $6/mo Buffer = ~$15/month total for daily multi-platform presence.

### Pinterest Strategy (High ROI for Travel)

Pinterest deserves special attention because:
- Users actively search for trip planning inspiration (high intent)
- Pins have 3-6 month lifespan (unlike social posts that die in hours)
- Travel is the #2 category on Pinterest
- Direct link-to-app from every pin

**Automated flow:**
```
[3x Weekly Cron]
  -> Claude Agent: generate 10 pin ideas from recent blog articles + destination data
  -> Gemini: generate tall (1000x1500) lifestyle images for each destination
  -> Claude Agent: write SEO-optimized pin titles and descriptions
  -> n8n posts to Pinterest via API
  -> Each pin links to app landing page with UTM tracking
```

**Target:** 30 pins/week. At 2% CTR on 10K monthly impressions = 200 site visits/week from Pinterest alone, growing as pin library compounds.

## 2. Video Content (Semi-Automated)

### AI-Generated Short Videos

**What it does:** Creates 15-60 second videos for TikTok, Instagram Reels, and YouTube Shorts — no filming required.

**How it works:**
```
[2x Weekly Cron]
  -> Claude Agent: write a video script (hook + 3 points + CTA)
  -> Text-to-speech: ElevenLabs or Google TTS generates voiceover
  -> Stock footage: Pexels API pulls matching travel b-roll
  -> n8n assembles via Creatomate API or Shotstack (programmatic video)
  -> Output: vertical video with text overlays, music, voiceover
  -> Queue in Buffer for posting
```

**Video types:**
- "3 things you didn't know about [Italian city]" — educational
- "POV: You open your anniversary trip app" — app walkthrough
- "What $10 gets you for your next trip" — value prop
- Italian pronunciation challenges — engagement bait

**Stack:** n8n + Claude API + ElevenLabs ($5/mo starter) + Creatomate ($12/mo) + Pexels (free)

**Cost:** ~$20/month for 8 videos/month. One viral video pays for years of this.

## 3. Customer Support (Fully Automated)

### In-App AI Concierge (Already Built)

You already have the Gemini-powered chat with Maps/Search grounding. This IS your customer support for trip-related questions. Extend it:

**Add these capabilities:**
- "How do I share my trip with family?" -> contextual help from a knowledge base
- "I can't see my partner's stamps" -> troubleshooting flow
- "How do I cancel?" -> retention offer ("Try it for one more trip free")
- Billing questions -> direct to Stripe customer portal link

**Implementation:** Add a system prompt layer that detects support-intent vs. travel-intent in the chat. Support questions get routed to a knowledge base RAG. Travel questions go to the existing grounded concierge.

**Cost:** Essentially free — it's the same Gemini API call you're already making.

### Email Support Agent

```
[On new support email via Helpscout/Freshdesk]
  -> Claude Agent: classify intent (billing / bug / feature request / general)
  -> Claude Agent: draft response using knowledge base + past resolutions
  -> If confidence > 90%: auto-send with "AI-assisted" footer
  -> If confidence < 90%: queue for Haris review (Slack notification)
```

**Expected volume:** At 5,000 users, ~50 support emails/month. AI handles 80% autonomously. Haris reviews ~10 emails/month.

**Stack:** Freshdesk free tier (up to 2 agents) + Claude API via n8n webhook

## 4. User Onboarding (Fully Automated)

### Smart Onboarding Sequence

When a new user signs up:

```
[Signup event webhook]
  -> Day 0: Welcome email with origin story + quick start guide
  -> Day 1: "Your AI itinerary is ready" email (pre-generated for their destination)
  -> Day 3: "Did you know you can share with family?" feature highlight
  -> Day 5: "Try today's Piazza Puzzle" engagement nudge
  -> Day 7: "Your trip countdown has begun" with countdown widget preview
  -> Day 14: If free tier, "Upgrade to Duo for $9.99" with comparison table
  -> Day 30: "Your trip is in X days!" urgency email
```

**All emails written by Claude, personalized with user's destination and trip dates.**

**Stack:** Resend ($0/mo for 3K emails/mo) or Loops.so ($0/mo for 1K contacts) + n8n triggers

### In-App Guided Tour

Already have the "Tap to flip" card hint. Extend to a 4-step interactive tour:
1. "This is your trip map" (highlight map)
2. "Flip the card to collect stamps" (highlight day card)
3. "Drag blocks to play the daily puzzle" (highlight puzzle)
4. "Share this link with family" (highlight share button)

Built with a lightweight tooltip system — no external dependency needed.

## 5. Sales & Conversion (Fully Automated)

### Abandoned Trip Recovery

```
[Daily Cron: check users who created a trip but didn't pay]
  -> Claude Agent: write personalized email based on their destination
     "Still dreaming of Tuscany? Your 7-day itinerary is waiting."
  -> Include a 20% discount code (auto-generated, single-use)
  -> Send via Resend
  -> If no conversion in 3 days, send final "trip leaves without you" email
```

**Expected lift:** 10-15% recovery rate on abandoned carts is standard. At 100 abandonments/month, that's 10-15 extra paid trips.

### Review & Testimonial Collection

```
[7 days after trip end date]
  -> Auto-email: "How was your Grand Tour?"
  -> One-click star rating in email
  -> If 4-5 stars: "Would you share a quick testimonial?"
  -> Claude Agent: take their raw feedback, polish into a testimonial (with approval)
  -> Auto-post to landing page testimonials section
  -> If 1-3 stars: "We're sorry. What could we improve?" -> route to support
```

## 6. Destination Content Generation (Fully Automated)

### Auto-Expanding the Destination Library

The biggest bottleneck to scaling is destination coverage. Currently: Italy only. Need: 50+ destinations.

```
[On demand or weekly batch]
  -> Input: destination name (e.g., "Kyoto, Japan, 5 days, romantic")
  -> Claude Agent with Gemini grounding:
     1. Generate 5-day itinerary with stops, restaurants, hotels
     2. Add drive/transit times between stops
     3. Generate parking/transport warnings
     4. Write milestone quotes per day
     5. Create daily reveal content (30 tiles)
     6. Generate Italian-equivalent local phrases (Japanese phrases for Kyoto)
     7. Write itinerary context for AI concierge
  -> Output: JSON matching TripSegment[] schema
  -> Review queue: Haris spot-checks 1-2 destinations/week
  -> Auto-publish approved destinations to the app
```

**Cost:** ~$0.50 per destination in API tokens. 50 destinations = $25 total. One-time cost, infinite reuse.

**Quality check:** The first 10 destinations should be hand-reviewed carefully. After that, the AI learns the pattern and quality stabilizes. Spot-check 10% ongoing.

## 7. Product Iteration (AI-Assisted)

### Bug Reports -> Fixes

```
[User reports bug via in-app feedback or email]
  -> Claude Agent: classify severity (critical / high / medium / low)
  -> Claude Agent: search codebase for likely cause
  -> Claude Code: propose a fix as a PR draft
  -> Haris: review PR, approve or reject
  -> Auto-merge and deploy on approval
```

This is literally what you're doing right now with Claude Code. Formalize it as a pipeline.

### Feature Requests -> Specs

```
[Aggregate feature requests weekly]
  -> Claude Agent: cluster similar requests, rank by frequency
  -> Claude Agent: write a 1-page spec for the top request
  -> Haris: approve or defer
  -> Claude Code: implement approved specs
```

## 8. Analytics & Reporting (Fully Automated)

### Weekly Business Report

```
[Every Monday 8 AM]
  -> n8n: pull data from Stripe (revenue), Supabase (signups, trips),
     Plausible/Umami (traffic), Buffer (social engagement)
  -> Claude Agent: write a narrative summary
     "This week: 47 new signups (+12% WoW), 18 paid trips ($179.82 revenue),
      top traffic source was Pinterest (34%), blog article 'Tuscany Guide'
      drove 200 visits. Recommendation: double down on Tuscany content."
  -> Send via email or Slack to Haris
```

**Haris reads a 2-minute summary instead of checking 5 dashboards.**

### Anomaly Detection

```
[Hourly health check]
  -> n8n: check API error rates, payment failures, signup drops
  -> If anomaly detected: Slack alert to Haris with context and suggested action
  -> If critical (site down, payments failing): auto-page via PagerDuty/SMS
```

## 10. What Haris Actually Does Each Day

### Daily (15 minutes)
- Check Slack for any flagged items (support escalations, content approvals)
- Skim the auto-generated social posts in Buffer queue, approve batch
- Glance at revenue dashboard on phone

### Weekly (1 hour)
- Read the Monday business report
- Review 2-3 blog article drafts, approve or give one-line feedback
- Spot-check one new destination the AI generated
- Make one strategic decision if needed ("let's target honeymoon keywords this month")

### Monthly (2 hours)
- Review financial summary
- Approve/reject AI-proposed feature specs
- Record one 60-second "face of the brand" video if desired (optional, AI can do faceless)
- Adjust pricing or promotions if needed

**Total time commitment: ~3-4 hours/week.**
